# Petty Cash Management System

A multi-user petty cash tracking system with role-based access control.

## Roles
- **Admin** — Allocates petty cash to employees
- **Employee** — Submits petty cash expenses
- **Accountant** — Views reports and analytics

## Tech Stack
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Frontend:** React, Tailwind CSS, Axios
